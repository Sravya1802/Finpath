# FinPath Mobile - React App

A mobile-first React application for financial guidance for U.S. newcomers.

## Features

### High Priority Features:
1. **Bank Comparison** - Compare and select eligible bank options based on visa type
2. **Safe Spending Calculator** - Calculate safe credit card spending limits
3. **Critical Alerts** - View upcoming deadlines and important reminders
4. **Money Transfer** - Compare safest and cost-effective money transfer services
5. **Tax Information** - Get tax filing information based on visa type

## Installation

```bash
npm install
```

## Running the App

```bash
npm start
```

The app will open at `http://localhost:3000`

## Mobile-First Design

The app is designed for mobile devices with:
- Bottom navigation bar
- Touch-friendly buttons and cards
- Responsive layout
- Mobile-optimized spacing and typography

## Project Structure

```
src/

    BottomNav.js        # Navigation component
```
